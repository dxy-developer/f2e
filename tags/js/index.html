最近看到挺多次 Rollup 这个词，再也架不住好奇，简单的学习实践了一下。完整项目库地址请戳。

PS: ES6 对应 ES2015，请忽略这些细节。

什么是 Rollup

Rollup Github 地址。根据官网的解释，Rollup 是下一代 ES6 模块化工具。ES6 之后，模块化的写法将更加的趋势化，我们会将以前的文件切割成多个的细小模块。那么如何来高效的组织管理这些文件，又有了很多不同的方案。现有的模块化打包已经有如 Browserify 和 Webpack ，那为啥还需要一个新的呢？

优势在哪

  • 可以生成 AMD，CMD，UMD 甚至 ES6 模块文件。
  • Tree-shaking
